# Smoking-Risk-Analytics-PowerBI

Power BI dashboard for analyzing smoking habits, lung health, cholesterol and hypertension risks through interactive visualizations and healthcare analytics.

---

## Project Overview

This project focuses on analyzing smoking-related health data using Power BI to generate meaningful insights into smoking behavior, lung health conditions, cholesterol levels, hypertension risks, and overall patient health trends.

The dashboard enables healthcare professionals and analysts to understand smoking patterns, identify health risks, and support data-driven healthcare decision-making through interactive visualizations.

---

## Objectives

* Analyze smoking habits across different age groups
* Monitor lung health conditions and risk factors
* Evaluate cholesterol and hypertension risks
* Compare smoking status by gender
* Build interactive healthcare dashboards
* Generate actionable insights through data visualization

---

## Dataset Information

The project uses healthcare datasets containing patient demographics, smoking habits, health conditions, and risk indicators.

| Dataset            | Description                     |
| ------------------ | ------------------------------- |
| health_dataset.csv | Main healthcare dataset         |
| condition.csv      | Health condition lookup dataset |
| Organs.csv         | Organ mapping dataset           |
| Image Dataset.csv  | Medical image mapping dataset   |
